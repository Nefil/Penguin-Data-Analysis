A short and simple project for exploring the Palmer Penguins collection. This README is a concise quick start guide.

## What's here
- Notebooks (.ipynb) with analysis and simple modeling.
- Supporting scripts in `scripts/` (optional).

## Quick Start
1. Clone the repo:
```bash
git clone https://github.com/Nefil/Penguin-Data-Analysis.git
cd Penguin-Data-Analysis
```

2. Create and activate the environment and install the dependencies:
```bash
python -m venv venv
source venv/bin/activate # macOS/Linux
.\venv\Scripts\activate # Windows (PowerShell)
pip install -r requirements.txt
```

3. Launch Jupyter:
```bash
jupyter lab
```

## IMPORTANT - Data File Path
This project does NOT contain a default data file path. You must provide your own path before running the notebook/script.

Minimum Requirements
- Python 3.8+
- pandas, numpy, matplotlib, seaborn, scikit-learn, jupyterlab
